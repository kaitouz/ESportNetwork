# UI

## List of interfaces

#### The pages will be available for guest (unregistered users)
+ Home Page
+ Login
+ Registration

#### The pages will be available for actor __*user*__
+ Home Page (as user)
+ User's profile
+ Edit User's profile
+ Profiles of other users/coaches
+ Search Course
+ Course Information

#### The pages will be available for actor __*coach*__
+ Home Page (as coach)
+ Coach's profile
+ Edit Coach's profile
+ Profiles of other users/coaches
+ Management Course
+ Search Course
+ Course Information
